== Description ==

This plugin will add the Piecemaker 3D flash slider to the Slider Manager's built-in slider types when using a WordPress theme with the Theme Blvd framework version 2.0.4+.

== Installation ==

1. Upload `themeblvd-piecemaker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure the `xml` folder within the plugin folder has been given writeable permissions (777).
